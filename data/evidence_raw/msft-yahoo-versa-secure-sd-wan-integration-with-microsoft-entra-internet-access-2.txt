SANTA CLARA, Calif., October 21, 2025--Versa, a global leader in unified networking and security, today announced the availability of an automated integration of Versa Secure SD-WAN with Microsoft Entra Internet Access, a component of Microsoft’s Security Service Edge (SSE) solution, to create a complete, easy-to-manage SASE solution that delivers comprehensive security with optimized performance for today’s branch offices.
